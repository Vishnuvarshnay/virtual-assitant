# 🎙️ AI Virtual Assistant with Cloud Integration
[![Live App](https://img.shields.io/badge/Render-Live_Application-green?style=for-the-badge&logo=render)](https://virtual-assitant-i5r9.onrender.com/signup)
[![GitHub Code](https://img.shields.io/badge/GitHub-Repository-blue?style=for-the-badge&logo=github)](https://github.com/Vishnuvarshnay/virtual-assitant)

An intelligent, voice-enabled digital assistant designed for web automation and media management. This project bridges the gap between **Natural Language Interaction** and **Cloud-based Services**.

## 🌐 Live Resources
- **Live Application:** [Click here to visit](https://virtual-assitant-i5r9.onrender.com/signup)
- **Primary Backend:** Deployed on **Render** with secure environment management.

## 🚀 Key Technical Highlights
- **Hybrid Backend Architecture:** Leveraged **FastAPI** (Python) for high-performance AI tasks and **Node.js/Express** for robust user management.
- **Voice Automation Engine:** Integrated Speech-to-Text (STT) and Text-to-Speech (TTS) capabilities to automate tasks like YouTube searches, weather updates, and system commands.
- **Cloud-Native Media Handling:** Implemented **Cloudinary** and **Multer** for secure storage and efficient retrieval of user-generated media assets.
- **Secure Session Management:** Built an authentication system using **JWT** to maintain persistent user context and personalized assistant settings.

## 🛠️ Main Features
- **Voice-Driven Web Tasks:** Commands like "Open YouTube" or "Search for [topic]" automated via voice.
- **Cloud Storage Integration:** Real-time profile management and media uploads powered by Cloudinary.
- **Interactive UI/UX:** Responsive interface built for seamless voice-interaction.
- **Modular Design:** Professional folder structure for high maintainability and scalability.

## 💻 Tech Stack & Tools
- **Languages:** JavaScript (Node.js), Python (FastAPI)
- **Database:** MongoDB (Mongoose)
- **Security:** JWT, Bcrypt
- **Cloud & Deployment:** Cloudinary, Render

## ⚙️ Quick Setup
1. Clone: `git clone https://github.com/Vishnuvarshnay/virtual-assitant.git`
2. Install: `npm install` & `pip install -r requirements.txt`
3. Configure `.env` for Cloudinary/MongoDB.
4. Run: `npm run dev`
